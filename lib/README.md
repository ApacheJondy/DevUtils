# Lib

该目录主要存储 DevUtils project 全部 lib module，封装快捷使用的工具类及 API 方法调用，隐藏内部逻辑判断对外提供便捷辅助类、统一回调类、Bean、Event 以及 Engine 兼容框架等


## 目录结构

```
- lib                        | 根目录
   - DevApp                  | Android 工具类库
   - DevAssist               | 封装逻辑代码, 实现多个快捷功能辅助类、以及 Engine 兼容框架等
   - DevJava                 | Java 工具类库 (不依赖 android api)
```
